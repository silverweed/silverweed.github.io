Hacknid is a puzzle stealth game developed in 48 hours during the Global Game Jam in 2018.

Infiltrate an enemy base and avoid being seen by remote controlling the robot guards. Pay attention: all commands you give to the guards are executed by all of them simultaneously!

Play with a controller or keyboard + mouse.

Left Joystick | WASD
Move the character
LB / RB | Mouse hover
Select previous / next command
A | Mouse click
Issue the selected command
B | R
Restart the level
